## elephant_pot system
* 基本规则
> 容易random(10, 15)

> 中等random(5, 10)

> 困难random(0, 5)
* 每天0点之前未提交算作没有完成
* 补交：第二天0点之前连同当天的一共两份都交上去可以减去之前一次的结果
* 每个自然月最多有**三次**补交机会
