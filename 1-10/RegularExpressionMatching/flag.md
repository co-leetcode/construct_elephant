# 动态规划

## 此题的动态方程

```
    1.p[j] == s[i]:dp[i][j] = dp[i-1][j-1]

    2.p[j] == ".":dp[i][j] = dp[i-1][j-1]

    3.p[j] =="*":

        3.1 p[j-1] != s[i]:dp[i][j] = dp[i][j-2]

        3.2 p[i-1] == s[i] or p[i-1] == ".":

        dp[i][j] = dp[i-1][j] // 多个a的情况

        or dp[i][j] = dp[i][j-1] // 单个a的情况

        or dp[i][j] = dp[i][j-2] // 没有a的情况
```
